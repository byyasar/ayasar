Bu projedeki amacım script keyboard yapmak.
* Projede işlemci olarak Stm32f103 Bleupill kullandım.
* 10 Adet Cherry Mx tuş kullandım.
* 1 Adet 1.3 inc SH1106 oled kullandım. 
* 1 Adet KY-040 Rotary Encoder kullandım.
